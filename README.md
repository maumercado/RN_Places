# RN Places

A React Native mobile application for exploring device features like camera, location, and permissions management. Practice project for working with native device capabilities.

## Features

Access device camera to take photos
Request and manage location permissions
Display user location with geolocation
Store and retrieve place information
Handle runtime permissions
Save photos and location data

## Requirements

Node.js (specified in .nvmrc)
React Native development environment
iOS Simulator or Android Emulator
Physical device for testing camera and location features

## Installation

Clone the repository
Install dependencies:
```
npm install
```

Start the development server:
```
npm start
```

Run on iOS or Android following React Native CLI instructions

## Technologies

React Native
JavaScript
Expo (if applicable)
React Navigation
Native device APIs (Camera, Geolocation)
